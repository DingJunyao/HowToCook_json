# 详细脂肪酸营养素说明

## 概述

USDA SR Legacy 数据库包含非常详细的脂肪酸分类，这些营养素键名遵循特定的命名规则。这些数据反映了食物中不同类型脂肪酸的详细组成。

---

## 命名规则解释

### MUFA (Monounsaturated Fatty Acids - 单不饱和脂肪酸)

格式：`mufa_<碳原子数量>:<双键位置>`

| 键名 | 说明 | 双键位置 |
|--------|------|---------|
| `:1` | 单键 | 单键位置 1（通常是末端） |
| `:1_c` | 顺式 | 顺式双键（常见于植物油脂） |

**常见 MUFA：**
- `mufa_16:1` - 棕榈酸
- `mufa_18:1` - 油酸
- `mufa_18:1_c` - 顺式油酸

### PUFA (Polyunsaturated Fatty Acids - 多不饱和脂肪酸)

格式：`pufa_<碳原子数量>:<双键位置>_<同分异构体>`

| 后缀 | 说明 |
|------|------|
| `:_n_3` | 3-顺式 | 双键在碳原子 3 位置，顺式（常见于 DHA/EPA） |
| `:_n_6_cc` | 6-反式 | 双键在碳原子 6 位置，反式 |
| `:_t` | 反式 | Trans fat（反式脂肪酸，通常是不健康脂肪） |
| `:_c` | 顺式 | Cis fat（顺式脂肪酸） |

**常见 PUFA：**
- `pufa_18:2_n_3_cc` - DHA（二十二碳六烯酸，顺式，对大脑健康重要）
- `pufa_18:3_n_3_cc` - DHA（二十二碳六烯酸，顺式）
- `pufa_18:2_n_3_t` - 反式DHA
- `pufa_18:3_n_3_t` - 反式DHA
- `pufa_20:5` - EPA（二十碳五烯酸）

**EPA/DPA：**
- EPA（二十碳五烯酸）：20:5
- DPA（二十二碳六烯酸）：22:6_n_3
- DPA (n-3)：22:6_n_6

### SFA (Saturated Fatty Acids - 饱和脂肪酸)

格式：`sfa_<碳原子数量>`

**常见 SFA：**
- `sfa_14:0` - 肉豆蔻酸（14个碳原子）
- `sfa_16:0` - 棕榈酸（16个碳原子）
- `sfa_18:0` - 硬脂酸（18个碳原子）

## 常见脂肪酸

### ω-3 系列（Omega-3）

这些是对人体健康最重要的多不饱和脂肪酸：

| EPA/DHA 名称 | 键名示例 | 主要来源 | 健康效益 |
|--------------|---------|---------|---------|
| EPA (二十碳五烯酸) | `pufa_20:5` | 深海鱼、鱼油 | 抗炎、心血管健康 |
| DHA (二十二碳六烯酸) | `pufa_22:6_n_3` | 深海鱼、藻油 | 大脑发育、视力健康 |
| α-亚麻酸 | `pufa_18:3_n_6_cc` | 植物油、坚果 | 心血管健康 |
| 亚麻酸 | `pufa_18:3` | 亚麻籽油 | 抗炎 |

### ω-6 系列（Omega-6）

| 名称 | 键名示例 | 主要来源 | 注意 |
|------|---------|---------|------|
| 亚油酸 | `pufa_18:2` | 植物油 | 过多可能促炎 |
| γ-亚麻酸 | `pufa_18:3` | 植物油 | - |

## 健康影响

### 单不饱和脂肪酸 (MUFA)
- **对健康有益**：改善胆固醇水平
- **主要来源**：橄榄油、坚果、鳄梨
- **代表**：`mufa_18:1`（油酸）、`mufa_18:1_c`（顺式油酸）

### 多不饱和脂肪酸 (PUFA)
- **ω-3 (Omega-3)**：抗炎、对心血管和大脑健康重要
  - EPA、DHA 主要来自深海鱼
  - α-亚麻酸、亚麻酸主要来自植物油

- **ω-6 (Omega-6)**：适量需要，过多可能促炎
  - 亚油酸主要来自植物油

### 饱和脂肪酸 (SFA)
- **过多摄入可能增加**：
  - 心血管疾病风险
  - 胆固醇水平升高

- **关键指标**：
  - `sfa_12:0`（月桂酸） - 椰物油主要成分
  - `sfa_14:0`（肉豆蔻酸）- 奶油、椰子油主要成分

## 在应用中的建议

### 1. 数据简化

**当前状态：**
这些详细的脂肪酸键名对于普通用户来说过于专业和复杂。

**建议方案：**

**选项A：合并为简化指标**
创建简化版：
- `omega_3_epa` - EPA 含量
- `omega_3_dha` - DHA 含量
- `omega_6` - 亚油酸含量
- `omega_3_total` - ω-3 总量
- `saturated_fat` - 饱和脂肪总量
- `monounsaturated_fat` - 单不饱和脂肪总量
- `polyunsaturated_fat` - 多不饱和脂肪总量
- `omega_3_ratio` - ω-3/ω-6 比值

**选项B：保留详细信息但添加映射**
在 `nutritions.json` 中添加一个 `nutrients_simplified` 字段，包含简化后的主要指标。

### 2. 可视化建议

在应用中显示这些数据时：
- **简化显示**：主要营养素（蛋白质、脂肪、碳水化合物、主要脂肪酸比例）
- **详细模式**：展开显示完整的脂肪酸分类
- **健康评分**：根据 ω-3/ω-6 比值和饱和/不饱和比例计算健康指数

### 3. 食谱筛选

根据营养数据提供筛选功能：
- **高 ω-3 菜谱**：适合心血管健康
- **低饱和脂肪**：适合控制胆固醇
- **健康脂肪比例**：ω-3/ω-6 > 2:1 为健康

## 示例数据转换

```python
def simplify_fatty_acids(nutrients):
    """简化脂肪酸数据为易于理解的指标"""
    simplified = {}

    # 总脂肪分类
    saturated = sum([
        nutrients.get('sfa_10:0', {}).get('value', 0),
        nutrients.get('sfa_12:0', {}).get('value', 0),
        nutrients.get('sfa_14:0', {}).get('value', 0),
        nutrients.get('sfa_16:0', {}).get('value', 0),
    ])

    monounsaturated = sum([
        nutrients.get('mufa_14:1', {}).get('value', 0),
        nutrients.get('mufa_16:1', {}).get('value', 0),
        nutrients.get('mufa_18:1', {}).get('value', 0),
        nutrients.get('mufa_20:1', {}).get('value', 0),
    ])

    polyunsaturated = sum([
        nutrients.get('pufa_18:2', {}).get('value', 0),
        nutrients.get('pufa_18:3', {}).get('value', 0),
        nutrients.get('pufa_20:5', {}).get('value', 0),
    ])

    # Omega-3 系列
    omega_3_epa = nutrients.get('pufa_20:5', {}).get('value', 0)
    omega_3_dha = nutrients.get('pufa_22:6_n_3', {}).get('value', 0)
    omega_3_total = omega_3_epa + omega_3_dha

    # Omega-6 系列
    omega_6 = nutrients.get('pufa_18:2', {}).get('value', 0) + nutrients.get('pufa_18:3', {}).get('value', 0)

    # 计算比例
    omega_3_ratio = omega_3_total / omega_6 if omega_6 > 0 else 0

    simplified['saturated_fat'] = saturated
    simplified['monounsaturated_fat'] = monounsaturated
    simplified['polyunsaturated_fat'] = polyunsaturated
    simplified['omega_3_epa'] = omega_3_epa
    simplified['omega_3_dha'] = omega_3_dha
    simplified['omega_3_total'] = omega_3_total
    simplified['omega_6'] = omega_6
    simplified['omega_3_ratio'] = omega_3_ratio

    # 健康评分（简单版）
    health_score = 0
    if 1 <= omega_3_ratio <= 2.1:
        health_score = 2  # 优秀
    elif 2.1 < omega_3_ratio <= 4:
        health_score = 1  # 良好
    elif 4 < omega_3_ratio <= 10:
        health_score = 0  # 一般
    else:
        health_score = -1  # 需要改善

    simplified['fat_quality_score'] = health_score

    return simplified
```

## 总结

1. **键名含义**：USDA 数据库使用专业的脂肪酸分类，准确反映食物的脂肪组成
2. **数据质量**：这些数据是准确的，但对于普通用户来说过于详细
3. **建议**：在应用中提供简化视图，计算健康指标
4. **价值**：这些数据支持精确的营养分析，适合专业营养计算和健康管理
